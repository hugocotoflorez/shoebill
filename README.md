```css
  --saira-red: #ffa3b6;
  --greedy-green: #d7ffbd;
  --golden-hermes: #ffffbd;

```

## Installation

It can be installed using lazy:

```lua 
{
        "hugocotoflorez/shoebill",
        lazy = false,
        priority = 1000,
},
```
